# OSDC Bot

![travis status](https://travis-ci.org/osdc/osdc-bot.svg?branch=master)

This bot lives at [https://gitter.im/osdc/Hackers](https://gitter.im/osdc/Hackers)

### Installation:

```
$ npm install
$ pip install -r requirements.txt
```

### Running:

```
$ node chatbot.js
$ python server.py
```

### Testing:

```
npm test
```
